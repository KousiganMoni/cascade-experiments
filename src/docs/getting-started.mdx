# Getting Started with PixelPulse

PixelPulse is a design system that helps you build consistent, accessible, and beautiful user interfaces.

## Installation

To install PixelPulse in your project:

```bash
npm install cascade
```

## Usage

Here's how to use PixelPulse components in your project:

```jsx
<Button variant="primary">Click Me</Button>
```
